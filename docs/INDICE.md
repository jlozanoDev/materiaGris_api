# MateriaGris API — Índice de Documentación

Este repositorio tiene dos tipos de documentación claramente separados:

| Tipo | Carpeta | ¿Qué contiene? | ¿Para quién? |
|------|---------|----------------|--------------|
| **Funcional** | [`docs/funcional/`](./funcional/INDICE.md) | Propósito de negocio, funcionalidades, reglas de negocio, flujos de API, perfiles, glosario | Product Owners, QA, nuevos miembros del equipo |
| **Técnica** | [`docs/tecnica/`](./tecnica/INDICE.md) | Arquitectura hexagonal, endpoints, modelo de datos, servicios, middleware, prompts de generación técnica | Desarrolladores |

---

## Navegación rápida
- [Documentación Funcional](./funcional/INDICE.md) — visión general, módulos funcionales, flujos, glosario
- [Documentación Técnica](./tecnica/INDICE.md) — arquitectura, módulos, endpoints, base de datos, gaps

---

## Regla de actualización

| Situación | Actualizar |
|-----------|------------|
| Nueva funcionalidad o cambio en requisitos de negocio | `docs/funcional/` |
| Cambio en implementación (endpoints, servicios, base de datos) | `docs/tecnica/` |
| Ambos | Ambas carpetas |
| Nuevo módulo | Añadir entrada en este índice maestro |
